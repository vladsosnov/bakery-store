# frontend

React + TypeScript app for Bakery Store.

## Structure

- `public/` static assets
- `src/app/` app bootstrap and routing
- `src/components/` reusable UI and layout components
- `src/features/` feature modules (catalog, cart, auth, chat)
- `src/hooks/` custom React hooks
- `src/lib/` utility helpers and shared config
- `src/services/` API clients and integrations
- `src/styles/` global styles and design tokens
- `src/types/` TypeScript types
- `tests/` frontend tests
